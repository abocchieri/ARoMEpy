# ``ARoMEpy``

## Introduction

``ARoMEpy`` is a Python implementation of the Rossiter-McLaughlin anomaly described in the ARoME code from [Bou√© et al. 2013](https://doi.org/10.1051/0004-6361/201220146)

## Installation

To install from source, clone the repository and move inside the directory.
Then use pip as:

```bash
pip install .
```

To install directly from the repository, use:

```bash
pip install git+https://github.com/abocchieri/ARoMEpy.git
```
